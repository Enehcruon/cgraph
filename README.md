CCGraph
===

> C Computation Graph Library

### Dependencies:

- LAPACK: `sudo apt-get install libblas-dev liblapack-dev`
- cmake `sudo apt-get install cmake`
- probably `build-essentials` as well.

###### Currently works on linux only.

### Limitations:
- Double numbers only.

## Future work
